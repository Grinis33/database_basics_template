@startuml
гість->система : реєструється на сайті
note right: Гість невірно заповнив поля анкети або залишив їх порожніми.
система->гість : надсилає запит підтвердження на почту
note right: Гість не підтвердив факт реєстрації.
гість->система :  підтверджує запит
система-> гість: створює особистий елктронний кабінет 
система->гість: отримує ідентефікаційний номер та пароль
